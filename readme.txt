動作環境
・Windows10
・Visual Studio 2017 (2015でも可）

開発言語：
C++

ビルド
cl /O2 AI.cpp

(clはvisual studio 2015以降のコンパイラ）

実行
AI.exe


入力など
・対戦回数　　…　連戦回数です。
・ポート番号　…　先手なら10000, 後手なら10001
・IPアドレス　…　サーバーのIPアドレスを指定します。同一PC上であれば127.0.0.1です。
・探索深さ　　…　一応設定できます。推奨値は6です。


実行例

対戦回数 ポート番号 IPアドレス 探索深さを入力↓
1 10000 127.0.0.1 6
